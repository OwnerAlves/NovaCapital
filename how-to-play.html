Como Jogar no Nova Capital RP

Siga este guia passo a passo para começar sua jornada em nosso servidor de SA-MP.

1. Requisitos Mínimos

  • Uma cópia do Grand Theft Auto: San Andreas (versão 1.0 US é frequentemente recomendada para maior compatibilidade com SA-MP e mods).
  • Cliente SA-MP (San Andreas Multiplayer) instalado no seu computador.
  • Microfone funcionando (essencial para comunicação por voz, caso o servidor utilize plugins para isso, ou para comunicação via Discord).
  • Conexão estável com a internet.
  • Ler e concordar com todas as regras do servidor.

Requisitos de Sistema (para GTA: San Andreas):

  • CPU: Intel Pentium IV ou AMD Athlon XP
  • GPU: Placa de vídeo compatível com DirectX 9 com pelo menos 128MB de VRAM (Nvidia GeForce 6 series ou ATI Radeon 9500 series ou superior)
  • RAM: 384MB (1GB+ recomendado para uma experiência mais fluida com mods)
  • HDD: Aproximadamente 4.7GB para a instalação do jogo base + espaço adicional para o cliente SA-MP e mods.

2. Instalação do SA-MP

Se você ainda não tem o SA-MP, siga estes passos:

  1. Certifique-se de que o Grand Theft Auto: San Andreas está instalado em seu computador. Se você possui a versão Steam ou Rockstar Games Launcher, pode ser necessário realizar um "downgrade" para a versão 1.0 US para garantir a compatibilidade com o SA-MP. Guias para isso podem ser encontrados online (ex: "downgrade GTA SA Steam for SA-MP").
  2. Acesse o site para download do Cliente SA-MP: Download Cliente SA-MP (ou seu sucessor oficial, como samp-multiplayer.com).
  3. Navegue até a seção "Downloads" e baixe a versão mais recente do cliente SA-MP (por exemplo, SA-MP 0.3.7-R4 Client ou similar).
  4. Execute o arquivo de instalação do SA-MP que você baixou.
  5. O instalador solicitará que você indique o diretório onde o GTA: San Andreas está instalado. Selecione a pasta correta.
  6. Siga as instruções na tela para concluir a instalação.

3. Junte-se ao nosso Discord

O Discord é essencial para nossa comunidade. Lá você encontrará anúncios, atualizações, suporte e poderá interagir com outros jogadores.

Entrar no Discord

No Discord, familiarize-se com os canais, especialmente #regras e #anuncios.

4. Como Entrar no Servidor

Após instalar o SA-MP, siga estes passos para conectar-se ao Nova Capital RP:

  1. Pelo Cliente SA-MP:
    • Abra o cliente SA-MP (procure por "samp.exe" na pasta de instalação do seu GTA: San Andreas).
    • No campo "Name" na parte superior do cliente, insira o nome de personagem que você usará no jogo (respeitando as regras do servidor sobre nomes).
    • Para adicionar nosso servidor: clique no botão laranja com um "tick" (✓ - Add Server) no canto superior esquerdo.
    • Uma pequena janela aparecerá. Insira o IP e Porta do nosso servidor: seu.ip.do.servidor:porta (substitua pelo IP e Porta reais fornecidos no Discord) e clique em "OK".
    • Nosso servidor aparecerá na sua lista de "Favorites". Selecione-o.
    • Clique no grande botão verde "Connect" (com um ícone de play) no canto superior esquerdo.
  2. Alternativa (Navegador de Servidores):
    • Você também pode tentar encontrar o servidor nas abas "Internet" ou "Hosted" do cliente SA-MP, mas adicionar aos favoritos é mais confiável.

Nosso Endereço (IP:Porta): seu.ip.do.servidor:porta (Este é um placeholder. O IP e Porta reais serão divulgados no Discord)

5. Primeiros Passos no Servidor

Ao entrar pela primeira vez, você provavelmente precisará registrar uma conta no servidor com uma senha. Dedique tempo para criar um personagem que se encaixe no ambiente de roleplay. Siga as instruções na tela e, se tiver dúvidas, procure ajuda no nosso Discord ou pergunte a jogadores mais experientes no jogo (de forma roleplay, claro!).

Lembre-se: leia atentamente todas as regras para garantir uma experiência divertida e justa para todos.
